# Polynomial Linear Regression Implementation 📈

This project demonstrates the implementation of a **Polynomial Linear Regression model** using Python and Machine Learning libraries. Polynomial Regression is used when the relationship between the independent variable and the dependent variable is **non-linear**.

## 📌 Project Overview

The objective of this project is to understand how **Polynomial Regression** can model complex relationships between variables by transforming input features into polynomial terms.

The code includes steps such as **data preprocessing, feature transformation, model training, prediction, and visualization.**

## ⚙️ Technologies Used

* Python
* NumPy
* Pandas
* Matplotlib
* Scikit-learn

## 📊 Steps Performed

1. Import required libraries
2. Load and explore the dataset
3. Apply **Polynomial Feature Transformation**
4. Train the **Linear Regression model** on polynomial features
5. Make predictions
6. Visualize the polynomial regression curve

## 📈 Output

The model fits a **curved regression line** instead of a straight line, allowing better predictions when the data has a **non-linear pattern**.

Graphs are used to visualize how the polynomial curve fits the dataset.
