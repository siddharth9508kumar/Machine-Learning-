# Multiple Linear Regression Implementation 📊

This project demonstrates the implementation of a **Multiple Linear Regression model** using Python and Machine Learning libraries. Unlike simple linear regression, multiple linear regression uses **two or more independent variables** to predict a target variable.

## 📌 Project Overview

The objective of this project is to understand how **Multiple Linear Regression** works by building a predictive model using multiple features from a dataset.

The code includes steps such as **data loading, preprocessing, model training, prediction, and evaluation.**

## ⚙️ Technologies Used

* Python
* NumPy
* Pandas
* Matplotlib
* Scikit-learn

## 📊 Steps Performed

1. Import required libraries
2. Load and explore the dataset
3. Handle and preprocess data
4. Select multiple independent variables (features)
5. Split dataset into **training and testing sets**
6. Train the **Multiple Linear Regression model**
7. Make predictions on test data
8. Evaluate model performance


## 📈 Output

The model predicts the target value using **multiple input features**.
This helps in understanding how different variables together influence the final prediction.
